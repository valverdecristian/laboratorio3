HTML BASICO

HTML no distingue entre minusculas y mayusculas, por lo que P es igual a p (para parrafos)

1 - Etiquetas

Tag: "todas las etiquetas tienen una función"
a: ruta de un archivo
b: texto en negrita
br: salto de linea (elemento vacio)
em: texto con énfasis
h1-h6: para titulos/encabezados y subtitulos. h1 es la mas importante
hr: define un salto temático en una página HTML
i: texto en itálica
mark: texto marcado
p: parrafo
strike: texto tachado
small: texto pequeño
strong: se resalta fuertemente el texto
title: titulo de página (general: va en el head)
u: texto subrayado
Las etiquetas que contienen a otras etiquetas son padres

4 - Enlaces/Hipervinculos

Permite vincular unas páginas con otras. Para esto usamos el elemento "a" con el atributo href
a: ruta de un archivo

Atributos:

href: Especifica la URL de la página a dirigirse
target: Especifica donde abrir el documento vinculado, algunas opciones son:
"_self": por defecto, se abre en la misma pestaña.
"_blank": para que se abra en una nueva pestaña, sin embargo puede tener implicaciones de seguridad, para mitigar riesgos de seguridad, es recomendable añadir 'rel=noopener noreferrer' al enlace.
download: Especifica cuál es el objeto que se descargará cuando un usuario haga clic en el hipervínculo.
title: Especifica información adicional sobre un elemento.

  • Enviar correo
Si queremos enlazar un archivo que esta dentro de una carpeta: carpeta/pagina.html
Para volver una carpeta anterior se usan dos puntos: ../pagina.html

Enlaces internos (anclas)

Se utiliza para indicar un elemento dentro de la misma página, dirigiéndose a un sector especifico. Para esto necesitamos el enlace propiamente dicho y el nombre (name) del ancla al cual debemos dirigirlo.

4 - Formularios

Se usa para pedir informacion/datos al usuario
form: define/construye un formulario. Una vez definido se debe indicar que tipo de inputs vamos a querer
input: lo usamos para entrada de datos

Atributos de input:

  • type="text" (permite tipear texto)
  • type="color" (permite seleccionar colores)
  • type="submit" (para enviar la informacion a un servidor)
  • type="file" (permite seleccionar un archivo)
  • y hay muchos otros type, depende que informacion requiera
  • required: campo requerido/obligatorio
  • name: identificador de los inputs (lo que le llega al servidor/nombre de campo: ej username)>
  • placeholder: escribe texto (por ejemplo la info que requerimos)
