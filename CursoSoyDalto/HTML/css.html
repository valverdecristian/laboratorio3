MODELO DE CAJA (box model)

  • Margin: espacio al rededor de la caja, que esta fuera del borde (no afecta a la caja)
  • Border: linea que rodea el contendio y el padding, se puede controlar el grosor, color y estilo del borde
  • Padding: espacio entre el contenido y el borde (relleno)
  • Content: area donde se muestra el contenido (texto/img)
  • border, padding y content afectan a la caja.
    • Atributos:
    • width: determina el ancho de la caja
    • height: determina el alto de la caja
    • top (arriba); bottom (abajo); left y right (izquierda y derecha)
    • se puede ahorrar estas cuatro lineas de código en 1 sola: ejemplo padding: 10px 0 20px 20px;
    • va en sentido de las agujas del reloj desde el top.
    • Al 0 no hace falta ponerle la unidad de medida
    • Si se pone unicamente dos valores, tomaria el 1er valor para top y bottom
    • "margin: auto"> centra horizontalmente

BORDES (Border)

  • border-width: modifica el tamaño
  • border-style: modifica el estilo
  • border-color: modifica el color
  • border: reemplaza a las 3 anteriores. (border: width style color)
  • border-radius:modificar las esquinas (propiedad aparte)
  • border-top/bottom/left/right
