comandos

para instalar npm se debe seleccionar la carpeta raiz y hacer click en "abrir terminal" y luego escribir el comando "npm init", luego "npm install bootstrap" y se va a descargar bootstrap y todas sus dependencias. Tambien debo instalar jquery "npm install jquery"

Si yo quiero compartir mi proyecto debo pasar el index.html y el package json unicamente. en la terminal escribir "npm install" y se van a descargar las dependecias [que estan en el packagejson]
